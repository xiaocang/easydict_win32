强大的 Windows 翻译词典应用，移植自广受好评的 macOS 应用 [Easydict](https://github.com/tisfeng/Easydict)。支持 18+ 种翻译服务、全局快捷键、划词翻译，界面简洁现代。

## 功能特性

- **多种翻译服务** — Google 翻译、Google 词典、必应翻译、DeepL、OpenAI、Gemini、DeepSeek、Groq、智谱 AI、GitHub Models、豆包、火山翻译、彩云小译、小牛翻译、Linguee 词典、Ollama、内置 AI 及自定义 OpenAI 兼容服务。多项服务无需 API 密钥即可使用。

- **LLM 流式翻译** — AI 服务翻译结果实时显示。

- **划词翻译** — 在任意应用中选中文本（拖拽、双击或三击），点击浮动翻译按钮即可在迷你窗口中翻译。

- **多种窗口模式** — 主窗口、紧凑迷你窗口、常驻固定窗口，适配不同使用场景。

- **全局快捷键** — 通过 `Ctrl+Alt+T`、`Ctrl+Alt+D`、`Ctrl+Alt+M` 等快捷键快速唤起。

- **剪贴板监听** — 自动翻译复制的文本。

- **深色 / 浅色主题** — 跟随 Windows 系统主题自动切换。

- **文字转语音** — 使用 Windows 语音合成朗读原文和译文发音。

- **HTTP 代理支持** — 为翻译服务配置代理服务器。

- **高 DPI 支持** — Per-Monitor V2 DPI 感知，在任意屏幕上清晰显示。

- **系统托盘** — 最小化到托盘后台运行。
