# Easydict.SidecarClient

A .NET library for IPC (Inter-Process Communication) with Easydict sidecar processes.

## Features

- **Process Management** - Start and manage sidecar processes
- **IPC Communication** - Communicate with sidecar via standard I/O
- **Async Support** - Full async/await support for non-blocking operations

## Installation

```bash
dotnet add package Easydict.SidecarClient
```

## Requirements

- .NET 8.0 or later
