Easydict for Windows is a free, open-source translation and dictionary tool with Copilot+ PC enhanced local AI translation.
- Copilot+ PC enhanced local AI — Phi Silica on Copilot+ PCs, Foundry Local on regular Windows PCs, OpenVINO NLLB-200 offline fallback
- 20+ translation services — Google, DeepL, Bing, OpenAI, Gemini, DeepSeek, Groq, Zhipu AI, and more
- Real-time AI streaming translation, OCR screenshot translation, PDF/Markdown long-document translation

On supported Copilot+ PCs with a 40+ TOPS NPU, Easydict translates with Phi Silica for fast, private, on-device output. Standard translation features remain fully available on every Windows device.

Easydict combines on-device local AI, traditional translation engines, and modern cloud AI models into a unified workflow — from quick word lookups to full document translation. It is a Windows port of the popular macOS Easydict app, built with modern .NET and WinUI 3 for a native experience.

Highlights:
- All-in-one translation: Google, DeepL, Bing and 20+ services
- Multi-AI support: OpenAI, Gemini, DeepSeek, Groq, Zhipu AI, and more
- OCR screenshot translation using native Windows OCR
- PDF and long-text translation with layout and formula preservation
- MDX custom dictionary support for advanced workflows
- Native Windows experience with instant selection translation
- Real-time AI streaming translation with multiple LLMs
- Multiple window modes, global hotkeys, clipboard monitoring, system tray
- Text-to-Speech, dark mode, high DPI displays

Easydict includes AI-generated translation and grammar-correction features. AI output may vary and should be reviewed by the user.
